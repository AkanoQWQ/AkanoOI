# 《不可以，总司令》

“根据最新消息，敌军摧毁了第45号、171号据点，我军依然顽强抗争，修复了据点37、98。现在可以反攻吗？”

“不可以，总司令。”

“敌军摧毁了第33号据点到42号据点的虫洞，现在可以反攻吗？”

“不可以，总司令。”

“我方修复了第42号据点，可以反攻吗？”

“不可以，总司令。”

“我方……”

“不可以，总司令。”

“同志，你只是一直在回答‘不可以’吗？你到底有没有好好判断形势？现在是关乎国家危亡的时刻……”

“总司令，您知道我不太聪明，没找到快速而正确地计算出结果的办法，但是据某项统计，我一直回答‘NO’的话，在一次战役中判断完全正确的概率是45%。”

“那……”

“不可以，总司令。”
